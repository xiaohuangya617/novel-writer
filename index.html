快速指南

✨ 快速开始 1. 新建小说 → 2. 点击⚙配置API Key → 3. 输入指令 → Ctrl+Enter 生成

⌨️ 全局快捷键

Ctrl+Enter 发送指令生成 · Ctrl+G 续写下一章 · Ctrl+F 查找 · Ctrl+H 替换 · Ctrl+Z/Y 撤销/重做 · Ctrl+S 保存章节

🎯 精准模式（新）

点击聊天区 🎯 按钮打开面板，勾选本章出场的角色和用到的动作。未勾选的设定不会注入 AI，大幅减少无关信息干扰。

🎯 创作逻辑链

系统提示：全局提示词 → 行文规范 → 角色关系图 → 角色设定 → 动作设定 → 服装设定 → 禁用词 → 世界观 → 情节主线 → 专有名词 → 输出

AI 严格遵循角色设定与动作规范，避免禁用词，保持前后文连贯。角色关系图全量注入，不受精准模式影响。

AI小说生成器 v3.1 完整说明

一、快速上手（3步）

1. 左侧面板点击「+ 新建小说」，输入书名创建

2. 点击聊天区右侧 ⚙ 按钮或顶部按钮，填入 DeepSeek API Key（需自行注册获取），点「测试连接」确认可用后「保存」

3. 在聊天框输入创作指令，按 Ctrl+Enter 或点击 ✈ 发送按钮，AI 自动生成章节

二、⌨️ 全部快捷键一览

Ctrl+Enter发送聊天指令，AI 开始生成新章节
Ctrl+G续写下一章（基于已有章节末尾）
Ctrl+F在当前章节中查找文本
Ctrl+H查找并替换文本
Ctrl+Z / Ctrl+Y撤销/重做编辑操作
Ctrl+A全选当前章节文本

三、🎯 精准模式（v1.5 新增，v1.7 优化）

点击聊天区右侧 🎯 按钮打开精准设定面板，控制 AI 生文时注入哪些角色和动作设定。

背景：默认全量注入角色库和动作库，大量无关设定稀释 AI 注意力。精准模式通过勾选，仅注入选中的角色和动作。

操作：点击🎯→左侧勾选本章出场角色→右侧勾选本章用到的动作→保存。按钮变绿表示精准模式开启。

校验：至少选1个角色或1个动作才能保存。点击「关闭精准模式」可恢复全量注入。

一次性（v1.7）：精准模式仅在 Ctrl+Enter 发送时生效，生完后自动关闭。续写和重写不受精准影响，始终全量注入。

建议：日常对话勾出场角色+「日常对话」动作；打斗勾对战双方+「打斗」；感情戏勾情侣+相关动作。

两种模式生文对比：

设定来源非精准模式精准模式
全局提示词 🌍✅ 全量注入✅ 全量注入
行文规范 ✍️✅ 全量注入✅ 全量注入
角色关系图 🔗✅ 全量注入✅ 全量注入（不受精准影响）
角色库 👥全部角色仅勾选的角色
动作库 💪全部动作仅勾选的动作
禁用词 🚫✅ 全量注入✅ 全量注入
世界观/情节/专有词 📖✅ 全量注入✅ 全量注入

四、📚 书架与章节管理

新建小说：左侧书架顶部「＋」图标或底部「＋新建小说」按钮。每本小说的数据独立存储。书架上每本小说卡片显示章节数、角色数和总字数。

切换小说：点击书架中的小说卡片即可切换，当前编辑内容会自动保存。

删除小说：点击小说卡片右侧 🗑 图标，需要二次确认。

章节列表：左下面板展示当前小说所有章节。点击章节名切换到该章；拖拽章节可重新排序；每章右侧有重命名 ✎ 和删除 🗑 按钮。

新增章节：章节列表顶部「＋」按钮，创建空白章节供手动书写。

五、✍️ 编辑器操作

◀ ▶ 按钮：切换到上一章/下一章

A+ / A-：调整编辑器字体大小（12px~24px），偏好自动记忆

🔍 查找：弹出查找栏，实时高亮所有匹配项，◀▶ 导航，支持替换单个或全部

💾 保存：手动保存当前章节到本地存储

续写：在已有最后章节的基础上，让 AI 自动生成下一章。可在聊天框输入额外指令（如"引入新角色"）来引导续写方向

重写：按原始指令重新生成当前章节。重写前自动保存快照，可用「回滚」恢复

回滚：撤销最近一次重写，恢复重写前的章节内容。仅在重写后可用

检查：调用 AI 对比前文内容与角色设定，检测外貌矛盾、时间线混乱、关系错误等潜在问题

六、💬 聊天指令区

位于页面中间的面板。输入创作指令（如「第二章，雨夜男主送女主回家，气氛暧昧」）后 Ctrl+Enter 发送。

生成时显示「✍️生成中...」，完成后章节自动保存并出现在编辑器和章节列表中。

聊天区最多保留 20 条历史记录，超出后自动移除最早的。可点击「清空记录」清除（有二次确认）。

每次生成完成后，聊天消息会显示 预估费用（按输入1元/百万token、输出2元/百万token计算）。

生成过程中可点击红色 ⏹ 停止 按钮取消当前生成。

七、📖 世界书设置（右侧面板）

📖 世界书 tab：

  • 世界观：故事发生的时代背景、地理设定、社会规则
  • 情节主线：故事的整体走向、核心冲突、关键转折
  • 专有名词：特殊词汇、人名地名、魔法体系等自定义术语及解释

🌍 全局提示 tab：

  • 全局提示词：定义 AI 的系统角色，如「你是一位擅长细腻情感的言情小说家，注重心理描写和氛围营造」
  • 行文规范：设定文风要求，如「文风细腻、多用比喻、段落简短有力」

以上所有文本框在失去焦点时自动保存。

八、👥 本书角色库

点击顶部「本书角色库」按钮打开管理面板。每个角色可设定：

  • 姓名：角色名称（必填）
  • 外貌体征：身高、发型、体型等
  • 性格：内向/外向、温柔/暴躁等
  • 服装及其他设定：常穿服饰、特殊能力、背景故事等
  • 身份关联：角色身份、与主线的关系等

角色设定在 AI 生文时作为系统提示注入。支持从「素材角色库」批量导入/导出。

九、🔗 本书关系图

点击顶部「本书关系图」打开 SVG 人物关系网络图。关系图数据会在 AI 生文时全量注入 system prompt（不受精准模式影响），以「角色关系图」段落形式出现在角色设定之前。

  • 添加关系：点击角色方块 → 在弹窗中选目标角色 + 输入关系名（如"母子""暗恋""宿敌"）→ 点添加
  • 删除关系：点击角色 → 在已有关系中点击 ✕ 删除
  • 颜色规则：同一种关系名自动分配同一种颜色，一目了然
  • 拖拽布局：可拖拽任意节点调整位置，点「重置布局」恢复环形排列

十、🎭 素材角色库

跨小说通用的角色模板库。可新增、编辑、删除素材角色，支持 JSON 导入/导出。在本书角色库中可一键批量导入素材角色。

十一、📝 素材草稿本（v1.8 新增）

顶部「素材草稿本」按钮打开独立文本编辑器，用于临时记录灵感、随笔、情节构思。内容存储在浏览器本地，点击「保存」生效。备份时会一并导出。

十二、💪 全局动作库

定义通用动作描写规范。每个动作包含：动作要领、台词运用、描写建议。AI 生成时会严格遵循这些规则来描写动作场景（如"打斗动作：注重速度和力量感，短句快速切换"）。支持 JSON 导入/导出。

十三、🚫 本书禁用词

添加不希望 AI 使用的词语或短语（如某些现代词汇、人名、俗语）。AI 在生成时会收到警告，尽力避免使用这些词汇。

十四、⚙️ API 调参

点击聊天区右侧 ⚙ 按钮打开设置：

  • API 地址 / API Key / 模型：必填项，默认指向 DeepSeek
  • 温度 (0~2)：越高输出越随机创意，越低越稳定保守
  • Top-p 核采样：控制词汇选择范围
  • 重复惩罚 / 频率惩罚 / 存在惩罚：控制 AI 重复自身的倾向
  • 上下文章节数：AI 生成时参考前多少章的内容（1~4章）
  • 停止词：AI 遇到这些词会停止生成（逗号分隔）

点击「测试连接」验证 API 可用性后再保存。

十五、💾 备份与导出

备份：导出包含所有小说、角色、动作库、API 配置的完整 JSON 文件。建议定期备份。

恢复：导入之前导出的备份 JSON，会覆盖当前全部数据，请谨慎操作。

导出全书：将当前小说的全部章节导出为 TXT 纯文本文件。

导出本章：仅导出当前编辑的章节为 TXT 文件。

十六、🛡️ 数据与安全

存储位置：所有数据存储在浏览器 localStorage 中，不会上传到任何服务器。清除浏览器数据会导致数据丢失。

容量上限：约 5MB，顶部实时显示使用量。超过 80% 时会变橙色警告，建议及时导出备份。

自动保存：每 30 秒自动保存一次；切换章节、切换小说、关闭页面时也会自动保存。

API Key 安全：仅存储在本地浏览器，仅用于调用 DeepSeek API，不会发送到其他地方。
