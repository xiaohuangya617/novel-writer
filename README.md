# AI小说生成器

基于 DeepSeek API 的智能小说创作工具。

**在线使用**：https://xiaohuangya617.github.io/novel-writer/

## 特色

- 🎯 **精准模式** - 只注入勾选的角色和动作，无关设定不干扰 AI
- 🔗 **关系图谱** - 可视化角色关系，AI 自动遵循
- 📚 **多书管理** - 书架独立存储，章节拖拽排序
- 💪 **动作库** - 自定义动作规范，AI 严格遵循
- 🔍 **冲突检测** - AI 自动检查剧情矛盾
- 💾 **本地存储** - 数据不上传，可备份恢复

## 数据安全

- 所有数据存储在**你的浏览器本地**，不会上传到任何服务器
- API Key 仅保存在你的电脑上，仅用于调用 DeepSeek API
- ⚠️ 清除浏览器缓存会导致数据丢失，请定期点击「备份」导出
